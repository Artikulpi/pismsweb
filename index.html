Manajemen Penggunaan Aplikasi pisms

Untuk menggunakan aplikasi,
user harus login terlebih dahulu,
setiap user dapat mengakses fitur-fitur SMS,
adapun pengaturan user hanya dapat diakses oleh user Admin.™ PISMS

User Level :
A. General User User biasa, dapat mengelola SMS.
B. Admin User Super user, dapat mengelola SMS dan dapat mengelola User.

Manajemen Kontak

Pengguna dapat mengelola kontak,
menambah, memperbarui, dan menghapus kontak, adapun informasi kontak sebagai berikut :

1. Nama (mandatory)
2. No. HP (mandatory)
3. Organisasi (optional)
4. Group (optional)

Kontak dapat dikelompokkan berdasarkan Group;
Setiap kontak dapat masuk ke dalam lebih dari satu group; Edit

Manajemen pesan

-Inbox
Pengguna dapat membaca SMS masuk;
-Outbox

Pengguna dapat mengirimkan SMS, terdapat beberapa moda pengiriman SMS :
a. SMS ke nomor di luar kontak, dengan satu atau lebih nomor tujuan
b. SMS ke nomor kontak, pengguna dapat memilih nomor dari kontak, atau melalui group (SMS broadcast)
Pengiriman melalui group dapat lebih dari satu group.

-Sent Items
Pengguna dapat melihat pesan sudah dikirim.
